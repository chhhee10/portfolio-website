Windows 95 Portfolio

A Windows 95 inspired personal portfolio for Chetan Raghuvanshi, built with React and a retro desktop-style interface.

Highlights

- Windows 95 style desktop experience
- About, Projects, Resume, Mail, and utility windows
- Interactive taskbar and Start menu
- Retro games embedded directly in the page
- Custom project directory and project detail windows
- Resume, GitHub, and LinkedIn shortcuts

Tech Stack

- React
- Framer Motion
- React Draggable
- Vite

Notes

- The portfolio is customized around Chetan's personal information, projects, and links.
- Static assets used for the UI are included in `src/assets`.
